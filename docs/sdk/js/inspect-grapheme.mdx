---
title: inspectGrapheme()
---

## Usage

```ts
const contractAddress = "";
const tokenId = "";

const report = await nameguard.inspectGrapheme("𝒶");
```

## Options

You can override the network used for `inspectGrapheme` using the second argument:

```ts
const contractAddress = "";
const tokenId = "";

const report = await nameguard.inspectGrapheme("𝒶", {
  network: "...",
});
```
